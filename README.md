# 🤖 Robottino Amico — WebAR Experience

Esperienza WebAR sviluppata come parte della tesi magistrale in Intelligenza Artificiale presso IULM Milano.

## 🔗 Link

👉 **[Apri l'esperienza AR](https://rosy15bhi.github.io/robottino-amico-ar/)**

## 📌 Descrizione

Questo progetto presenta **Robottino Amico** in realtà aumentata — il nucleo cognitivo di un futuro robot fisico progettato come estensione della mente dell'utente.

La pagina utilizza **Google Model Viewer** per renderizzare il modello 3D direttamente nel browser. Su dispositivi mobili è disponibile il tasto **"Vedi in AR"** per proiettare il robottino nella stanza reale.

## 🛠️ Tecnologie

- [Google Model Viewer](https://modelviewer.dev/) v3.4.0
- GitHub Pages
- WebXR / Scene Viewer / Quick Look (AR nativa iOS e Android)
